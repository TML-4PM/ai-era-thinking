# AI Value Gradient Research v1

## Purpose
Identify where machine efficiency collapses cost while human-perceived value remains high or increases.

## Core Model
Human–Machine Value Inversion Curve.

Axes:
- X: Machine cost (high → low)
- Y: Human perceived value (low → high)

## Zones
1. Commodity Automation
   - Low perceived value, low cost.
   - Race to zero.

2. Premium Human Zone
   - High perceived value, high cost.
   - Still human-led.

3. Sweet Spot (Target)
   - Low machine cost, high human value.
   - Immediate build priority.

## Key Equation
Value = Context + Timing + Emotion + Experience

## Initial Domains
- Sport
- Events
- Education
- Mobility
- Workflows

## Hypothesis
The best products are not where AI is strongest, but where humans still care deeply while machines can deliver cheaply.

## Deliverable
Map all T4H businesses and candidate products into this matrix and identify the highest-value crossover zones.
